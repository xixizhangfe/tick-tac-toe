# tick-tac-toe
a small game by React, suitable for beginner!

## How to use
#### 1、clone to your computer

	git clone https://github.com/xixizhangfe/tick-tac-toe.git

#### 2、enter the directory

	cd tick-tac-toe
	
#### 3、install packages

	npm install 
	
#### 4、run
	
	npm start
	
#### 5、You will see the result
